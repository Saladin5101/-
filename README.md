# 系统日志自动告警邮件发送脚本(适用于macOS)
本脚本推荐适用于macOS15及以上版本,其余版本未经测试

## 功能:
本脚本在启动后,可以在监控到警告和错误两个级别的日志后向计算机所有者发送告警邮件,通知所有者持续监控他们的设备,以防止突发事件

## 开源协议:
本仓库使用GPLv3协议保护其代码,敬请留意相关内容
